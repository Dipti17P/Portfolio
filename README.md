# 🌌 Dipti Patil — Full-Stack & AI Developer Portfolio

An extraordinary, highly interactive, and responsive developer portfolio for **Dipti Patil**, specializing in Backend Systems, Machine Learning Pipelines, and Full-Stack Web Development. This application features state-of-the-art UI elements, an immersive cosmic-themed particle background, and fluid, scroll-triggered animations.

---

## ⚡ Key Highlights
- **Cosmic-Infused Theme**: High-contrast, ambient dark layout with smooth, interactive particle networks tracking the user's cursor.
- **Scroll-Triggered Section Entrances**: Smooth slide-up animations built with `motion/react` that trigger gracefully when sections scroll into view.
- **Aesthetic Typography**: Styled using custom, modern typeface pairings, generous negative space, and vibrant neon accents (`neon-cyan` and `neon-purple`).
- **Interactive Project Cards**: 3D hover effects with real-time mouse-tracking spotlights, metrics displays, and quick-launch links.
- **Optimized for Portability**: Native responsive design conforming to desktop precision and thumb-friendly mobile layouts.

---

## 🛠️ Technology Stack

| Layer | Tools & Technologies |
|---|---|
| **Frontend UI** | React, TypeScript, Tailwind CSS, Lucide Icons |
| **Animation Engine** | `motion/react` (formerly framer-motion) |
| **Interactive Assets** | HTML5 Canvas API (custom particles) |
| **Package Bundling** | Vite, Esbuild |
| **Linting & Code Quality** | ESLint, TypeScript Compiler (`tsc`) |

## 📁 Project Structure

```bash
├── src/
│   ├── App.tsx                  # Root component coordinating scroll-anchors and app layout
│   ├── main.tsx                 # Core bundle and React DOM entry point
│   ├── index.css                # Global styles, Tailwind directives, custom grid-patterns
│   ├── data.ts                  # Central database of projects, experiences, credentials, and skills
│   └── components/
│       ├── Navbar.tsx           # Floating navigation bar with active section highlighting
│       ├── Hero.tsx             # Typographic splash screen and interactive summary
│       ├── About.tsx            # Personal background, education, and credentials
│       ├── Skills.tsx           # Animated database categories and technology tags
│       ├── Experience.tsx       # Interactive responsive work timeline
│       ├── Projects.tsx         # Responsive list container of full-stack achievements
│       ├── ProjectCard.tsx      # Advanced 3D hover-tilt card with dynamic spotlights
│       ├── Achievements.tsx     # Visual ledger showcasing certifications and qualifications
│       ├── Contact.tsx          # Copyable email, social portals, and responsive form
│       ├── ParticleBackground.tsx # High-performance interactive HTML5 canvas particle generator
│       ├── ScrollCounter.tsx    # Scroll percentage overlay
│       └── Preloader.tsx        # Styled developer preloader with booting sequence animation
├── metadata.json                # Project description and major capabilities
├── package.json                 # Project dependencies and deployment scripts
└── tsconfig.json                # TypeScript typing configurations
```

## 🚀 Local Development

To run this project locally, execute the following commands in your terminal:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Boot the development server:**
   ```bash
   npm run dev
   ```

3. **Compile and build for production:**
   ```bash
   npm run build
   ```

4. **Lint code and analyze type safety:**
   ```bash
   npm run lint
   ```
